一：介绍
  1、这是用tensorflow写的图像分类程序，本程序只是一个示例醒程序，只是对apple和orange进行分类
  2、该程序的特点是从头开始一点一点的，详细的完成了整个项目
  
二：如何使用
  1、准备数据集
    1.1、下载图片，在项目路径下新建image/train和image/test两个文件夹，并将下载的图片分别保存到对应的文件夹，在放入train和test目录下
    1.2、之多tfrecords文件，参考load_data.py文件
    
  2、搭建网络(参考train.py文件)
    2.1第一步，先使用read_and_decode(）方法读入数据
    2.2第二步，搭建网络
    2.3第三步，将数据传入网络进行训练
    
三、注意
    1、读取图片和训练是分两个线程，如未开读数据线程，则程序会一直为start状态，但无法进行sess.run()
    2、在准备数据集的时候，注意图片的大小，如果图片过大，则可能会内存溢出，造成训练n个epoch之后无法进行
